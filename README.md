# robot-arm-cv


A  robotic hand tracking visualization built with **Python**, **OpenCV**, and **MediaPipe**. The application detects a user's hand through a webcam and overlays a futuristic robotic HUD, animated joints, tracking effects, and cyberpunk-style graphics.

## Features

- Real-time hand tracking using MediaPipe
- Robotic hand skeleton visualization
- Animated mechanical joints and palm reactor
- Fingertip motion trails
- Hand rotation and palm data display
- Cyberpunk HUD interface
- Dynamic arm rendering
- Smooth motion interpolation and tracking
- Live FPS monitoring

## Requirements

- Python 3.8+
- Webcam
- Internet connection (first run only)
